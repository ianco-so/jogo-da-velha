Tic-Tac-Toe Documentation

Overview

This project is a web-based implementation of the Tic-Tac-Toe game. It allows players to choose the board size and play in two modes: Human vs Human or Computer vs Human.

Functionalities Implemented

  • Dynamic board size selection (3x3 to 10x10).
  • Game modes: Human vs Human, Computer vs Human.
  • Switch between light and dark themes.
  • Player vs Player and Player vs Computer interactions.
  • Win and draw conditions with appropriate messages.

Functionalities Not Implemented

There are no functionalities that were not implemented according to the specifications.

Reflection

During the development process, we learned the importance of clear project organization and collaboration. If we were to redo the project, we might improve the structure of our code for better modularity.
