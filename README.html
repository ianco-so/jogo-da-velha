Documentação: Jogo da Velha

Resumo

Este projeto é uma implementação web do Jogo da Velha.
Ele permite que os jogadores decidam o tamanho do
tabuleiro e se o jogo irá acontecer no modo Humano x Humano
ou no modo Máquina x Humano.

Funcionalidades implementadas

  • Seleção do tamanho dos tabuleiros (de 3x3 até 10x10)
  • Modos de jogo: Humano vs Humano, Máquina vs Humano.
  • O jogo decide automaticamente quem foi o vencedor ou se houve empate

Funcionalidades não implementadas

  • Seleção de modo claro ou escuro
  • Um modo máquina mais competitivo
  • Registro dos jogos já jogados

Reflexões

O projeto foi bem divertido de se fazer, mas a minha falta de domínio sobre o JavaScript
me fez criar funções verbosas e com pouca modularidade, como a função ehVitoria.

Rodando o projeto

Para rodar o projeto, você precisa garantir que os arquivos index.html,
script.js e style.css estejam na mesma pasta.
Daí basta abrir o arquivo index.html em qualquer navegador web.
